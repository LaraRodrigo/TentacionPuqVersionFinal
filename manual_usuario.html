Manual de Usuario

Este sitio web va dirigido para los usuarios finales como para los administradores del sitio web.

Requisitos del sistema

Desde cualquier dispositivo, ya sea celular, tablet, notebook o un computador de escritorio, se puede ejecutar sin problemas el sitio web, solo se necesita que tenga conexión a internet.

Acceso al sitio web

Este sitio web puede ser encontrado desde la url: (página local que se corre desde los equipos) localhost/index.html


Los requisitos para iniciar sesión a este sitio web son los siguientes: tener un correo electrónico ya sea Gmail o Outlook, tener contraseña, crear su nombre de usuario.


Para moverse en este sitio web se creó un motor de búsqueda con las palabras claves: Chaquetas, Parkas, Polera, Poleras, Jockey, Jockeys, etc.


El apartado de menú desplegable nos da la posibilidad de movernos más rápidos por los diferentes productos que se ofrecen. También existe un carrusel el cual nos deja ver productos de manera rápida dándonos un vistazo más amplio de estos.


Login / registro aquí se puede tanto si ya tienes una cuenta como si quieres crearte una cuenta para poder estar en un modo "Administrador" y un cliente "Común" el cual será dado por el tipo de correo que se utilice.

Funcionalidades Principales

Una de las funcionalidades principales de este sitio web es que se tiene un motor de búsqueda para dirigirse a las distintas secciones del sitio haciendo que sea más rápido encontrar los distintos productos que se están vendiendo en esta empresa, dándoles a los usuarios una mejor manera de encontrar productos que les interese. Otra funcionalidad que está en el prototipo sería el apartado de registro y login el cual tiene la funcionalidad de subir las cuentas creadas a una base de datos la cual manejara los datos de los clientes para que sean reconocidos; el login es una funcionalidad que se creó a partir del registro de datos de nuestros módulos, haciendo capaz de consultar a la página si en la base de datos existe una persona con el Usuario llamado por ejemplo "Matías" y si es así déjame iniciar sesión con este nombre y una contraseña que fue "Hacheada" (hachear: encriptar la contraseña para mayor seguridad, ya que estos no se pueden "regresar a su estado original") para obtener seguridad en sus datos.

Glosario

Hachear: Hachear es un método de encriptación la cual sirve para darle seguridad a datos que deben ser "privados", como por ejemplo contraseñas y este método de encriptación no se puede retroceder, o sea, al momento de encriptar queda con esos mismos caracteres generados de forma aleatoria, pero la página al leer esa encriptación sabe que existe un dato que solo esta página mediante una llamada de llaves. Base de datos: es donde se almacenan los datos para un sitio web, aplicación, etc.
